# 트랙별 학습 경로

## 🟢 대학생 트랙
`00 AI 현실 → 01 생존 전략 → 02 DE 소개`로 동기를 잡은 뒤,
`03~06`에서 도구(터미널·Markdown·Linux·Git), `07~09`에서 Python,
`10~12`에서 SQL·Docker, `13~14`에서 데이터 처리·분석,
`15`에서 미니 프로젝트로 마무리합니다.

## 🔵 졸업생 트랙
`00 생존 전략 → 01 환경`으로 시작해 `02` 빠른 리프레셔 후,
`03~04` 모델링·SQL, `05~06` ETL·코드화, `07~08` 오케스트레이션·스트리밍,
`09~10` 품질·웨어하우스, `11` 캡스톤으로 마무리합니다.

## 트랙 연계
대학생 트랙 수료 → 졸업생 트랙 진입 시 `00~02`는 빠르게 통과 가능합니다.
