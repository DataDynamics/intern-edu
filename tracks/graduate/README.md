# 🔵 졸업생 트랙 — 실무 데이터 파이프라인 지향

> 기초 프로그래밍 경험이 있는 대학 졸업생을 위한 트랙입니다. 기초는 빠르게 복습하고 **실무 파이프라인 구현**에 집중합니다.

## 대상 & 선수지식
- **대상**: 대학 졸업생, 실무 투입 준비 단계
- **선수지식**: Python·SQL 기본 문법 경험 (부족하면 [대학생 트랙](../undergraduate/) 해당 모듈 참조)

## 목표
실제 **ETL/데이터 파이프라인**을 설계·구현하고, 오케스트레이션·데이터 품질·웨어하우스까지 모던 데이터 스택을 이해합니다.

## 진행 방식
- 모듈을 순서대로(`00 → 11`) 진행하되, 실습 비중이 높습니다.
- 00은 작성·토론 중심(`worksheet/`), 01부터 코드·파이프라인 실습입니다.
- 매 모듈의 산출물이 쌓여 마지막 **캡스톤 프로젝트**의 토대가 됩니다.
