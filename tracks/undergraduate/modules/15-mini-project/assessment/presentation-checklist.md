# 발표 체크리스트 (5분)

- [ ] **문제**: 어떤 데이터를, 무슨 질문으로 분석했는가 (30초)
- [ ] **데이터 품질**: 어떤 문제가 있었고 어떻게 정제했는가 (1분)
- [ ] **분석 결과**: 핵심 발견 1\~3가지 + 차트 (2분)
- [ ] **해석**: "그래서 무엇을 시사하는가" (1분)
- [ ] **회고**: 배운 점, 어려웠던 점, AI 도구 활용·검증 경험 (30초)

## 발표 팁
- 코드를 줄줄 읽지 말고 **결과와 인사이트** 중심으로
- 차트 1장으로 핵심을 말할 수 있게 준비
- "왜 그렇게 정제했는지" 질문에 답할 수 있도록
