# 데이터 흐름 지도 그리기

익숙한 서비스 하나를 골라, 그 안의 데이터가 어떻게 흐를지 상상해 그려보세요.
정답은 없습니다. "데이터의 눈"으로 서비스를 보는 연습입니다.

## 1. 고른 서비스
> 예: 배달앱 / 음악 스트리밍 / 쇼핑몰 ...

## 2. 어떤 데이터가 생기는가? (수집)
- 사용자가 만드는 데이터 (예: 주문, 검색, 클릭):
- 자동으로 쌓이는 데이터 (예: 로그, 위치):

## 3. 어디에 저장될까? (저장)
- (예: 주문 DB, 로그 저장소 ...)

## 4. 어떻게 가공될까? (변환)
- (예: 일별 매출 집계, 추천을 위한 사용자 행동 정리 ...)

## 5. 누가 어떻게 쓸까? (활용)
- (예: 사장님 대시보드, 추천 시스템, 분석가 리포트 ...)
