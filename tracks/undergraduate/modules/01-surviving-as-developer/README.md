## 🎯 학습 목표
- AI를 대체재가 아닌 **증폭기**로 쓰는 원칙을 세운다
- 대체되지 않는 핵심 역량이 무엇인지 안다
- AI 시대에 맞는 학습법을 익힌다
- 자신만의 12주 성장 계획서를 작성한다

---

## 📚 핵심 내용

### 1. AI를 증폭기로 — 사람의 몫은 판단
- AI 코딩 도구(예: Claude Code 등)를 **페어 프로그래머**처럼 활용
- 단, **검증·판단·책임은 사람의 몫** — AI 답을 그대로 믿지 않는다
- "빠르게 만들고(AI) → 비판적으로 검증하고(사람) → 고친다" 의 반복

### 2. 대체되지 않는 4가지 역량
1. **문제 정의력** — 무엇을 풀지 정하는 능력 (AI는 시키는 것만 한다)
2. **데이터·도메인 이해** — 회사 고유의 맥락 (일반 AI가 모르는 영역)
3. **시스템 설계·디버깅** — 전체 그림을 잡고 문제를 추적하는 능력
4. **커뮤니케이션·협업** — Git/PR/리뷰로 함께 일하는 능력

### 3. T자형 인재
- 가로(─): 넓은 기초 (Linux·Git·Python·SQL·데이터)
- 세로(│): 깊은 한 축 — **데이터 엔지니어링**
- 이 교육은 정확히 이 T자를 만든다

### 4. AI 시대의 학습법
- 답을 외우지 말고 **"왜"** 를 묻는다
- AI 답변을 **비판적으로 검증**한다 (틀린 답·환각을 걸러내는 힘)
- 빠르게 만들어보고(prototype) 고치며 배운다
- 배운 것을 **GitHub에 기록**해 "AI가 아니라 내가 한 사고 과정"을 증명한다

### 5. 포트폴리오·기록의 힘
- 결과물(코드·프로젝트·리포트)을 GitHub에 남긴다
- 깔끔한 커밋 이력 = 꾸준함과 사고 과정의 증거
- 이 교육의 모든 실습·프로젝트가 곧 포트폴리오가 된다
