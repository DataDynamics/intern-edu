# 실습 — 접속 로그 분석 스크립트 만들기

`data/access.log`(웹 서버 접속 기록)를 분석하는 쉘 스크립트 `analyze_log.sh`를 완성하세요.

## 요구사항
스크립트를 실행하면 아래 4가지를 출력해야 합니다.
1. **총 요청 수** (전체 줄 수)
2. **상태코드별 빈도** (예: 200이 몇 번, 404가 몇 번 …)
3. **에러 요청 수** (상태코드 400 이상)
4. **가장 많이 접속한 IP Top 5**

## 진행 방법
1. `analyze_log.sh`를 열어 `# TODO` 부분을 채웁니다.
2. 실행 권한을 줍니다: `chmod +x analyze_log.sh`
3. 실행합니다: `./analyze_log.sh`

## 힌트
- 총 줄 수: `wc -l`
- 상태코드는 각 줄의 **9번째 필드** → `awk '{print $9}'`
- IP는 각 줄의 **1번째 필드** → `awk '{print $1}'`
- 빈도 집계의 정석: `... | sort | uniq -c | sort -rn`
- 400 이상 필터: `awk '$9 >= 400'`
