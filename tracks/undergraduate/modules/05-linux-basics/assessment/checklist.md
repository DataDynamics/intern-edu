# 모듈 05 — 완료 체크리스트

학습자가 스스로 점검하고, 강사가 통과 여부를 확인합니다.

## 실습 통과 기준
- [ ] `examples/`의 스크립트 5개를 모두 실행해보았다
- [ ] `exercises/starter/analyze_log.sh`의 모든 TODO를 채웠다
- [ ] 스크립트에 `chmod +x`로 실행 권한을 주고 `./analyze_log.sh`로 실행했다
- [ ] 출력에 총 요청 수 / 상태코드 빈도 / 에러 수 / IP Top 5가 모두 나온다
- [ ] `assessment/quiz.md`를 풀고 정답과 비교했다

## 역량 확인 (강사 구두 점검)
- [ ] 파이프(`|`)가 무엇을 하는지 한 문장으로 설명할 수 있다
- [ ] `sort | uniq -c` 패턴을 왜 쓰는지 설명할 수 있다
- [ ] `>` 와 `>>` 의 차이를 안다
- [ ] 권한 숫자(예: 755)를 rwx로 풀어 설명할 수 있다
