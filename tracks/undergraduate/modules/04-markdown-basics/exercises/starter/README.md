# 실습 — Markdown으로 자기소개 문서 만들기

`profile.md`를 완성하세요. 아래 요소를 **모두** 포함해야 합니다.

| 요소 | 문법 |
|------|------|
| H1 제목 | `# 제목` |
| H2 제목 (2개 이상) | `## 섹션` |
| 굵게 | `**굵게**` |
| 순서 없는 목록 | `- 항목` |
| 순서 있는 목록 | `1. 항목` |
| 링크 | `[글자](URL)` |
| 인라인 코드 | `` `code` `` |
| 코드 블록 | ` ``` ... ``` ` |
| 표 | `\| .. \|` + 구분선 |
| 체크리스트 | `- [ ]` / `- [x]` |
| 인용문 | `> 내용` |

## 검증
```bash
python check.py
```
각 요소가 ✅ 로 표시되고 `✅ 모든 테스트 통과!` 가 나오면 성공입니다.
미리보기는 VSCode에서 `Cmd/Ctrl + Shift + V`.

> 정답 예시는 `../solution/profile.md` 참고 (먼저 스스로 작성해보세요!).
