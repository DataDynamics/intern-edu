# 정답 예시 — profile.md

`profile.md`가 요구된 11개 Markdown 요소를 모두 포함한 예시입니다.
중요한 건 문법을 **외우는 것**이 아니라, 필요할 때 치트시트를 보고 **빠르게 쓰는 것**입니다.

## 자주 하는 실수
- 표에서 헤더 아래 `|---|` 구분선을 빼먹음 → 표로 렌더링되지 않음
- 코드 블록 백틱을 2개만 쓰거나 닫지 않음
- `- [ ]` 체크리스트에서 대괄호 사이 공백을 빠뜨림 (`-[]`는 안 됨)
