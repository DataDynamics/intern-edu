# 모듈 04 — 퀴즈 정답

1. `**데이터 엔지니어**`
2. `## 설치 방법`
3. 대괄호 `[ ]` 에는 **표시할 글자**, 소괄호 `( )` 에는 **URL**.
4.
   ````markdown
   ```python
   print("hi")
   ```
   ````
5. 완료: `- [x] 항목` / 미완료: `- [ ] 항목`
6. 정렬/구분선 `|---|---|` (헤더와 본문을 나누는 줄). 없으면 표로 렌더링되지 않음.
7. 이미지는 앞에 **느낌표**가 붙는다: 링크 `[글자](url)`, 이미지 `![대체텍스트](url)`.
8. `#`=제목1(가장 큼), `##`=제목2, `###`=제목3. # 개수가 많을수록 하위 단계.
9. README 작성, PR 설명, 분석 리포트, 기술 문서, 이슈/위키 등 (2개 이상).
10. `Cmd/Ctrl + Shift + V` (또는 GitHub에 올리면 자동 렌더링).
