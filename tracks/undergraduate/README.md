# 🟢 대학생 트랙 — CS 신입, 완전 기초부터

> 컴퓨터공학을 이제 전공하려는 신입을 위한 트랙입니다. **프로그래밍 경험이 없어도** 따라올 수 있도록 설계되었습니다.

## 대상 & 선수지식
- **대상**: 컴퓨터공학 전공 예정/저학년 신입, 데이터 엔지니어 진로 탐색
- **선수지식**: 없음 (컴퓨터 기본 사용 가능 수준)

## 목표
AI 시대의 현실을 직시하고, 그 안에서 경쟁력을 가지는 **데이터 엔지니어 기본기**(Python · SQL · Linux · Docker · 데이터 처리)를 확보합니다.

## 진행 방식
- 모듈을 **순서대로**(`00 → 15`) 진행합니다.
- 각 모듈은 강의 문서(`README.md`)를 읽고 → `examples/`로 익히고 → `exercises/starter/`를 채운 뒤 → `assessment/`로 점검합니다.
- 모듈 00~02는 작성·토론 중심(`worksheet/`)이며, 03부터 실습 코드가 본격화됩니다.
